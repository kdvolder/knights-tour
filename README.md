Knights Tour
------------

An ocaml puzzle solver that solves the classic [Knight's Tour](https://en.wikipedia.org/wiki/Knight%27s_tour).

To run the solver:

```
dune exec ./solve_horsejump.exe
```

To play the game yourself interactively:

```
dune exec ./playground.exe
```
